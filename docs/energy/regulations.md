# Regulations (Spain)

## Applicable rules

| Rule | Scope | Key point |
|---|---|---|
| **RD 244/2019** | Self-consumption (autoconsumo) | Off-grid isolated systems are exempt from grid injection permits |
| **REBT RD 842/2002** | All low-voltage installations | Installation must be signed by a licensed electrician (*instalador autorizado*) |
| **ITC-BT-40** | Isolated generators and PV | Technical spec for isolated PV systems; covers protection and earthing |

## Off-grid vs. grid-tied

This system is designed **off-grid (aislado)**:
- No connection to the utility grid
- No net metering or feed-in tariff possible
- No permit needed to generate — only for the installation itself
- Must comply with REBT for safety of the installation

## Battery room requirements

Per REBT and fire code:
- Ventilation required (hydrogen off-gassing, even minimal with LiFePO4)
- No ignition sources within 0.5 m of battery bank
- Fire extinguisher CO₂ class within 5 m
- Temperature: keep below 40°C — install in a north-facing, ventilated room
